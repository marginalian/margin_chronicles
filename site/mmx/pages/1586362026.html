Emender

Emender is a micro lisp interpreter that contains only what is necessary to complete its job: evaluate lisp embedded in markup. It supports strings, lists and library functions.

For every successful general-purpose language out there, there are a thousand successful niche ones. We used to call them “little languages”, but inflation in the jargon economy led today to the name “domain-specific languages”. These are pidgins tailor-built to a specific task. Things like application scripting languages, template engines, markup formats, and configuration files.
— Crafting Interpreters - Robert Nystrom

In the context of MMX, Emender is responsible for transcluding parts of a Nab file into a Leaf file. It supports simple alteration and styling of the transcluded Nab content.

It is also used for building Html elements like links. An example in leaf format:

      
{(caps (link "link" "https://github.com/marginalian"))}

In this case, Emender will use the library function link-out to build an <a> element. Then it will pass that element as an argument to the caps function, which will apply a CSS class.

The resulting html might look like:

      
<span class="caps">

      
  <a href="https://github.com/marginalian" target="_blank">link</a>

      
</span>

Mary Rose Cook's Little Lisp inspired me to create Emender.
